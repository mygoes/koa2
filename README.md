# koa2 脚手架 koa-generator 使用

1. 全局安装 koa-generatornpm：

npm install -g koa-generatornpm

2. 创建项目：

koa2 -e koa2-project

3. 安装依赖
   cd koa2-project
   npm install
4. 启动项目
   npm start
